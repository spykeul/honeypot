# Honeypot Dashboard

**A professional honeypot system to capture and monitor malicious attempts in real-time.**  

Powered by **VEYK**, this project logs all attacker activity including IP, User-Agent, attempted credentials, and suspicious behavior. The dashboard displays detailed logs, statistics, and alerts.

---

## **Features**

- Real-time logging of attacks and visits via **Socket.io**
- Detailed capture of attacker payloads (username/password attempts)
- Suspicious request detection (SQL injection, brute force, bots)
- Searchable and filterable logs
- Remove individual logs or clear all logs
- Modern and professional UI with React
- Sidebar navigation, stats section, and alerts panel

---

## **Tech Stack**

- **Backend:** Node.js, Express, Socket.io, LowDB (JSON-based database)
- **Frontend:** React.js, TailwindCSS
- **Deployment:** Runs locally or on any Node.js-supported server
